# Practical Field Task Management App Requirements

## Form
- Auto-fill today's date
- Fields: Date, Reporter, Location, Status, Comment, Assignee (required), Due date

## Task Board UI
- Kanban style with 3 columns: Not Started, In Progress, Completed
- Each task is a card showing: Comment (title), Location, Assignee, Due date

## Usability
- Change status with one tap (cycle)
- Color coding: Red (Not Started), Yellow (In Progress), Green (Completed)

## Grouping
- Group tasks by date
- Show newest first

## Controls
- Add delete button to each task
- Show confirmation before delete

## Filters
- Show all
- Show only incomplete
- Show only overdue

## Mobile
- Responsive design
- Easy to tap buttons
- Clean and modern UI
